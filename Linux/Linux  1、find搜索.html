Linux 1、find搜索

发表于2019-06-29|更新于2023-08-02|Linux
|字数总计:950|阅读时长:3分钟|阅读量:

1.文件的查找

1.1按文件属性查找:

  • 文件名:find + 查找的目录 + -name + “文件的名字”

  • 文件大小: find + 查找目录 + -size + +10k

  • size默认单位：512B–》0.5k–》一个扇区的大小

  • 文件类型: find + 查找目录 + -type + d/f/b/c/s/p/l

文件类型 符号
普通文件 - (在搜索是用f)
目录 d
链接符号 l
块设备 b
字符设备 c
socket文件 s
管道 p(mkfifo创建管道)
  • 查找目录：find 目录 -maxdepth 层数 -type d

    depth 在查找文件时，首先查找当前目录中的文件，然后再在其子目录中查找。

2. 按文件内容查找:

grep -r “查找的内容” + 查找的路径
